# Інструкції по запуску Spotify WinForms

## Коротко

Проект запускається як звичайний Windows Forms застосунок на C# .NET 6.

## Що потрібно

- Windows
- .NET 6 SDK
- MySQL (якщо потрібен вхід/реєстрація)

## Запуск

1. Відкрити проект у Visual Studio або VS Code
2. Запустити `Spotify.WinForms`
3. За потреби додати музичні файли у папку `canciones`

## Що є в проєкті

- форма входу та реєстрації
- програвач музики
- улюблені композиції
- навчальна форма для обчислень

## Примітка

Плейлистів у проєкті немає — залишено лише реально використаний функціонал.
