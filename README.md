# Spotify WinForms

Невеликий віконний музичний застосунок на C# .NET 6 з Windows Forms.

## Можливості
- вхід і реєстрація користувача
- відтворення музичних файлів з папки canciones
- пауза та вибір треку
- управління улюбленими композиціями
- перегляд формул і навчального прикладу обчислень

## Технології
- C#
- .NET 6
- Windows Forms
- MySQL
- NAudio

## Запуск
1. Встановити .NET 6 SDK
2. Запустити проект у Visual Studio або через dotnet run

## Структура
- Models — моделі даних
- Controllers — робота з БД та файлами
- Views — форми застосунку

## Примітка
Проект спрощено: плейлистів і зайвого функціоналу немає, залишено лише те, що реально використовується.
