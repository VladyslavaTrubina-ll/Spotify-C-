# Spotify WinForms

Компактна віконна програма на C# .NET 6 з Windows Forms, що поєднує музичний плеєр і калькулятор для обчислення математичних функцій.

## Основні можливості

- **Аутентифікація користувача** — вхід та реєстрація через форму входу
- **Музичний модуль** — відтворення MP3/WAV файлів із локальної папки `canciones`
- **Керування плеєром** — кнопки Play, Pause, вибір треку зі списку
- **Модуль обчислень** — табуляція математичних функцій за заданими параметрами
- **Головне меню** — навігація між музикою та калькулятором

## Технологічний стек

- **Мова**: C#
- **Платформа**: .NET 6
- **UI**: Windows Forms (WinForms)
- **БД**: MySQL (ADO.NET)
- **Аудіо**: NAudio (відтворення)

## Опис інтерфейсу та вікон

### 1. Форма входу (Login)

![Логин](Foto%20README/Логин.png)

На цьому екрані користувач вводить облікові дані:

- **TextBox "Користувач"** — поле для введення логіна
- **TextBox "Пароль"** — поле з маскуванням символів для безпеки
- **Кнопка "Вхід"** — перевіряє дані в БД та відкриває головне меню
- **Кнопка "Реєстрація"** — переводить у режим реєстрації нового користувача

При помилковому вводі програма виводить повідомлення про необхідність коректного вводу.

---

### 2. Форма реєстрації (Registration)

![Регистрация](Foto%20README/Регистрация.png)

Аналогічна формі входу, але використовується для створення нового облікового запису:

- Користувач вводить бажаний логін і пароль
- При натисканні кнопки виконується перевірка унікальності логіна
- Успішна реєстрація дозволяє негайно увійти в систему

---

### 3. Головне меню (Main Menu)

![Меню](Foto%20README/Меню.png)

Центральний екран навігації програми, з якого відкриваються всі модулі:

- **Кнопка "Музика"** — перехід у музичний модуль для відтворення треків
- **Кнопка "Обчислення функції"** — перехід у модуль табуляції формул
- **Кнопка "Вийти"** — закриття програми та завершення сеансу

Меню оформлено у фірмовому стилі Spotify (темний фон, зелені кнопки).

---

### 4. Музичний модуль (Music Player)

![Музика](Foto%20README/Музика.png)

Форма для відтворення локальної музики:

- **ListBox зі списком треків** — відображає всі доступні MP3/WAV файли з папки `canciones`
- **Кнопка "▶ Play"** — запускає відтворення обраного треку
- **Кнопка "⏸ Pause"** — призупиняє поточне відтворення
- **Кнопка "← Назад"** — повернення до головного меню

Трек можна також запустити подвійним кліком по його назві в списку. Програма зчитує файли із папки при першому запуску музичного модуля.

---

### 5. Модуль обчислень (Function Calculator)

![Формули](Foto%20README/Формули.png)

Форма для табуляції математичних функцій:

- **Поля введення** (TextBox):
  - `xmin` — мінімальне значення аргументу
  - `xmax` — максимальне значення аргументу
  - `dx` — крок табуляції
  - Константа для формули (наприклад, `a`)
- **Кнопка обчислення** — запускає табуляцію функцій на заданому інтервалі
- **Списки результатів** (ListBox):
  - Перший список — результати для `f1(x)`
  - Другий список — результати для `f2(x)`

Програма виконує валідацію введених даних (перевірка діапазонів, типів, граничних випадків) та виводить помилки у MessageBox при невалідному введенні.

---

### 6. Додаткові вікна

![Окно1](Foto%20README/Окно1.png)

Під час роботи програма використовує:

- **MessageBox** — для інформаційних повідомлень, попереджень та помилок
- **Вікна результатів** — окремі ListBox для кожної функції, щоб не захаращувати екран

## Встановлення та запуск

### Вимоги

- **.NET 6 SDK** або новіше
- **Visual Studio 2022** (рекомендовано) або будь-яка C#-IDE
- **MySQL сервер** (для логіна та реєстрації)

### Кроки

1. **Клонування або розпакування проекту**

   ```bash
   cd Spoty
   ```

2. **Встановлення залежностей**

   ```bash
   cd Spotify.WinForms
   dotnet restore
   ```

3. **Налаштування БД**
   - Переконайтесь, що MySQL запущений
   - Імпортуйте схему з файлу `scriptDB/spoty.sql`
   - Переконайтесь, що у класі `ControladorDB` вказані коректні параметри підключення

4. **Створення папки для музики**
   - Створіть папку `canciones` у каталозі проекту
   - Додайте MP3 або WAV файли

5. **Запуск програми**

   ```bash
   dotnet run
   ```

   Або через Visual Studio: **F5** або **Debug → Start Debugging**

## Примітки

- **Спрощена архітектура**: проект зберігає тільки необхідний функціонал, без плейлистів, облюблених и зайвих функцій
- **Локалізація чисел**: при введенні параметрів можна використовувати як крапку, так і кому (залежно від налаштування системи)
- **Обробка помилок**: валідація на рівні форм, MessageBox для зворотного зв'язку
- **Многопоточність**: відтворення музики працює в окремому потоці, щоб не блокувати інтерфейс
